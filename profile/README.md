# Macs Fan Control Utility

**Macs Fan Control Utility** is a powerful fan management application for Mac users who want complete control over their system's cooling. As a precise **Macs Fan Speed Manager**, you can set fan speeds to a fixed RPM or use auto mode based on any temperature sensor (CPU, GPU, SSD, battery, or ambient). This **Macs Temperature Monitor** displays real-time readings from multiple sensors directly in your menu bar with customizable color alerts for overheating. With **Macs Fan Curve Tool**, you can create custom fan response curves that automatically increase fan speed when specific components reach user-defined temperature thresholds.

## Features

- **Full Control**: The **Macs Fan Control Utility** works on all Intel and Apple Silicon Macs.
- **Sensor Tracking**: Use **Macs Fan Speed Manager** to see CPU die, CPU proximity, GPU, and disk temps.
- **Silent Mode**: As a **Macs Temperature Monitor**, it allows aggressive or quiet cooling profiles.
- **Custom Curves**: This **Macs Fan Curve Tool** supports hysteresis to prevent fan pulsing.
